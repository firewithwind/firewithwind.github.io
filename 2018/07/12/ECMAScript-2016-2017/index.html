ECMAScript 2016-2017

2016:

Array.prototype.includes(value)

value: any type
return: Boolean

幂运算符(**)

example:

4 ** 2
// 16

2017:

async and await

SharedArrayBuffer

Object.entries 和 Object.values

Object.entries 返回由键值对组成的数组，不会返回原型链上的属性

Object.values 返回由对象的值构成的数组，不包含原型链属性

String.padStart 和 String.padEnd

字符串补全

接收两个参数，第一个是最终的位数，第二个是用来补全的模板

example:

'123'.padStart(6, 'ab')
// 'aba123'
'123'.padEnd(7, 'abc')
// '123abca'

Object.getOwnPropertyDescriptors

该函数接收一个对象，并将对象的值转化为对象的值的描述

参数中的尾随逗号合法化
